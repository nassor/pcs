Nassor is the creator and primary author of Canudo. Senior Rust developer building a high-performance async workflow orchestration library. Comfortable with advanced Rust patterns (async_trait, blanket impls, trait objects, Arc/RwLock). Values API ergonomics and clean architecture -- evidenced by the Task/Node duality design and comprehensive examples. Published on crates.io (v0.7.2).
